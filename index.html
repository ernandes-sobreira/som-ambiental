Teste de Normalidade (Shapiro-Wilk aproximado)

O teste de Shapiro-Wilk avalia se os dados seguem uma distribuição normal (p > 0,05 = normal). Com amostras pequenas (n < 5), o resultado é indicativo e deve ser interpretado com cautela. Se os dados forem normais, use teste t ou ANOVA para comparações; caso contrário, prefira testes não-paramétricos (Mann-Whitney, Kruskal-Wallis).
